# 【开源推荐】LomenTV VDS - 专为 Android TV 打造的 WebDAV 视频播放器

> 一款支持 WebDAV 网盘、智能刮削、记忆续播的 Android TV 视频播放器，已开源！

## 简介

LomenTV VDS 是一款专为 Android TV 和电视盒子设计的视频播放器应用。它支持连接飞牛 NAS 的 WebDAV 服务，自动刮削媒体元数据，提供媲美专业流媒体平台的观影体验。

## 核心功能

### 📁 WebDAV 网盘支持
- 完美支持飞牛 NAS 的 WebDAV 服务
- 支持 OpenList、AList 等 WebDAV 服务器
- 自动扫描媒体文件，智能分类整理

### 🎬 智能媒体刮削
- **本地优先**：优先读取本地 .nfo 文件，保护你的自定义元数据
- **TMDB 刮削**：自动从 TMDB 获取海报、简介、评分等信息
- **多类型支持**：电影、电视剧、纪录片、演唱会自动分类

### 📺 TV 端专属优化
- 遥控器方向键焦点导航
- 沉浸式全屏播放体验
- 针对大屏优化的卡片式 UI

### ⚡ 播放功能
- **记忆续播**：自动记录播放进度，下次从断点继续
- **智能跳过**：支持跳过片头片尾
- **音轨/字幕**：播放时自由切换

### 🔧 其他特性
- 首页滚动通知（远程可配置）
- 最近观看记录
- 媒体搜索功能
- 自定义 TMDB API Key

## 系统要求

- Android 8.0 (API 26) 及以上
- Android TV 或电视盒子
- 飞牛 NAS 或其他 WebDAV 服务器

## 快速开始

### 1. 下载安装
从 GitHub Releases 下载最新 APK 安装包：
https://github.com/jia070310/lomenTV-VDS/releases

### 2. 配置 WebDAV
1. 打开应用，进入设置页
2. 选择"添加 WebDAV 网盘"
3. 填写飞牛 NAS 的 WebDAV 地址：
   - 地址：`http://你的飞牛IP:5005`
   - 路径：`/dav/你的文件夹`
   - 用户名/密码：你的飞牛账号

### 3. 等待刮削
应用会自动扫描并刮削媒体信息，稍后即可在首页看到精美的海报墙。

## 技术亮点

- **Kotlin + Jetpack Compose TV**：现代化 TV 应用开发
- **MVVM 架构**：代码清晰，易于维护
- **Room 数据库**：本地数据持久化
- **智能刮削引擎**：剧集聚类，减少 API 调用
- **GitHub 加速**：通知文件通过 gh-proxy.org 加速访问

## 开发者说

这款应用最初是为了方便在电视上看飞牛 NAS 里的电影而开发的。相比传统的 SMB/NFS 挂载方式，WebDAV 更加稳定，而且支持外网访问。

开源的目的是希望能帮助到更多有同样需求的用户，也欢迎开发者一起完善这个项目。
